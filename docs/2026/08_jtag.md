# Настройка JTAG-программатора

1. Проверить, что встроенный в плату JTAG-программатор обнаруживается в `lsusb`.
2. Отключить программатор.
3. Установить udev-правила для программатора, запустив с правами администратора скрипт `data/xicom/cable_drivers/lin64/install_script/install_drivers` из каталога с установленной Vivado.
4. Перезагрузить ПК.
